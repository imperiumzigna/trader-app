### Setup

The app uses a Makefile to setup the project.

#### Install dependencies

```
$ make init
```

#### Start dev environment

```
$ make start
```

#### Run container console

```
$ make shell
```

#### Linting

```
$ make lint <file or folder> -- -a
```

#### Testing

```
$ make shell
$ bundle exec rspec ...
```
